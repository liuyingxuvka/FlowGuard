## Why

Guard-family skills can stop after partial work when missing inputs, stale evidence, skipped checks, or child-Guard gaps are only described in prose. FlowGuard needs a shared closure report shape so it can route those gaps as maintenance obligations before broad confidence claims.

## What Changes

- Add a Guard closure contract validator for child Guard reports.
- Teach the FlowGuard kernel to treat child Guard partial/blocked/downgraded reports as maintenance obligations.
- Require latest-schema FlowGuard project maintenance before skill completion claims.

## Capabilities

### New Capabilities
- `guard-closure-contract`: Shared machine-readable child-Guard closure reports and FlowGuard validation.

### Modified Capabilities
- None.

## Impact

Affected surfaces: `model-first-function-flow` skill guidance and assets used by Guard-family orchestration.
